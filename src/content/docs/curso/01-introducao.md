O **Claude Code** é uma ferramenta de linha de comando criada pela Anthropic que conecta a IA diretamente ao seu ambiente de desenvolvimento — sem copiar e colar, sem trocar de janela.

## Claude.ai vs Claude Code

| | Claude.ai (chat) | Claude Code (CLI) |
|---|---|---|
| Onde roda | Navegador | Terminal do seu computador |
| Acesso aos seus arquivos | Não | Sim — lê e escreve no seu projeto |
| Executa comandos | Não | Sim — roda testes, instala pacotes |
| Commita no Git | Não | Sim |
| Memória do projeto | Não | Sim, via `CLAUDE.md` |

Em resumo: o Claude.ai é um chat. O Claude Code é um colaborador que vive dentro do seu projeto.

## O que você vai conseguir fazer

Ao final deste curso você será capaz de:

- Abrir qualquer projeto no VS Code e chamar o Claude para trabalhar com você
- Pedir ao Claude para criar arquivos, funções, componentes e páginas inteiras
- Fazer deploy de um site ou app com um único comando
- Montar agentes de IA que executam tarefas automaticamente
- Integrar APIs externas no seu projeto sem escrever todo o código na mão
- Usar MCP servers para dar ao Claude acesso ao browser, email e outros serviços

## Pré-requisitos

Você não precisa saber programar. Mas precisa ter:

- Um computador com macOS, Windows ou Linux
- Acesso à internet
- Uma conta na [Anthropic Console](https://console.anthropic.com) (gratuita para criar)
- Vontade de escrever instruções em português ou inglês
